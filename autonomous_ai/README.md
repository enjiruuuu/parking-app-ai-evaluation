# autonomous_ai

A SolidJS application for the autonomous AI parking app.

## Getting Started

This project is built with SolidJS for better Playwright integration.

- [Learn SolidJS](https://www.solidjs.com/docs)
- [SolidJS tutorials](https://www.solidjs.com/tutorial/introduction)

## Development

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```
